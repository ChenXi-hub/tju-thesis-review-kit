## 仓库首页简介

`TJU Thesis Review Kit` 是一套面向天津大学本科毕业设计（论文）的开源辅助工具包，包含 Cursor `Skill` / `Rule`、可直接复制的 prompt、`PROJECT.md` / `MEMORY.md` 模板，以及一个可公开分发的 `LaTeX` starter 模板。它的目标不是替代导师和学院要求，而是帮助同学们更系统地完成论文自查、格式整改和送审前 review。

## 仓库亮点文案

- 把零散的毕设经验整理成可复用的 Cursor 能力
- 兼顾学校规范、论文语体、图表检查和送审前总检
- 不只给规则，也给 starter 模板和可直接复制的 prompt
- 适合“从写作到终稿 review”的完整流程
