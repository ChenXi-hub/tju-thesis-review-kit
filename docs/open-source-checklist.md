# Open Source Checklist

发布这套仓库前，建议你再确认下面几项：

- [ ] 选择并补充 `LICENSE`
- [ ] 阅读并确认 `THIRD_PARTY.md` 的表述符合你的预期
- [ ] 仓库首页写清适用范围：天津大学本科毕业设计（论文）
- [ ] README 中写清“仍需以学院最新要求为准”
- [ ] 确认没有残留个人姓名、学号、导师、课题名、私有链接或本地路径
- [ ] 确认 LaTeX 模板不依赖你本机独有字体和图片
- [ ] 确认示例参考文献、摘要、正文均为占位内容
- [ ] 检查 `docs/release/github-copy.md` 与 `docs/release/github-release-draft.md` 是否符合你想公开表达的口径
- [ ] 如准备长期维护，增加版本记录或更新日期
