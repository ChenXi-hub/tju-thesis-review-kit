# Contributing

欢迎补充和改进这套仓库。

## 适合提交的内容

- 天津大学本科毕设 review 规则的补充与纠错
- 更稳健的 Cursor `Skill` / `Rule`
- 更适合本科论文场景的 prompt 模板
- 更简洁、可复用的 LaTeX starter 改进
- README、示例和文档完善

## 不建议直接提交的内容

- 个人课题内容、个人姓名学号、导师姓名、私有路径
- 某一专业过强绑定的写法，除非明确拆成独立扩展
- 编译产物，例如 `.aux`、`.log`、`.toc`、`.pdf`
- 未说明来源的第三方模板文件替换

## 提交建议

- 尽量保持通用性，不要把规则写死在某一篇论文上
- 对模板或样式文件的大改动，请说明修改动机和适用范围
- 若涉及第三方来源，请同时更新 `THIRD_PARTY.md`
- 若新增 prompt 或模板，优先保证“能直接复制使用”

## Issue 建议

提 issue 时，最好说明：

- 你遇到的问题属于格式、图表、摘要、参考文献还是模板编译
- 你使用的是哪个学院或哪一版学校要求
- 你期望仓库提供的是规则、模板还是 prompt
