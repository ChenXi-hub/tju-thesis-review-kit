# Recommended Prompts

以下问法可直接复制到 Cursor 中使用。

## 1. 全文格式初审

```text
请按天津大学本科毕业设计（论文）规范 review 我的 LaTeX 仓库。

重点检查：
1. 前置页顺序
2. 页码切换
3. 页眉页脚
4. 标题层级
5. 摘要与关键词

请按“问题清单（按严重程度）—已修改文件—剩余风险—编译结果”的结构输出。
```

## 2. 图表专项整改

```text
请重点检查这篇论文中的图表是否符合天津大学本科毕业论文要求。

重点看：
1. 图题在下、表题在上
2. 图表编号是否按章
3. 表格是否为三线表
4. 图中文字、图例、单位是否清晰统一
5. 图注和表注是否使用送审语气，避免“作者自绘”“脚本路径”等痕迹

如果能直接修改，请直接改；如果不能，请给出按优先级排序的问题清单。
```

## 3. 摘要与语体检查

```text
请检查我的中文摘要、英文摘要和关键词，重点看：
1. 中英文是否对应
2. 是否有内部文件名、本地路径、过程描述
3. 是否存在明显的 AI 套话或答辩备忘口吻
4. 是否适合送审

请直接指出问题并给出可替换的修改版本。
```

## 4. 参考文献核对

```text
请检查正文引用和 references/ref.bib 是否一一对应，并重点看：
1. 是否有未定义引用
2. 是否有文中说了作者但 cite 键不一致
3. 图表来源是否可追溯
4. 文献体例是否统一

请先列问题，再说明建议如何改。
```

## 5. 送审前总检

```text
请对我的论文做送审前最后一轮总检。

要求：
1. 优先找会影响提交的硬问题
2. 再看图表、摘要、参考文献和附录
3. 如果能编译，请编译并报告 warning
4. 输出一个可直接发给导师的问题清单

输出结构：
1. 问题清单（按严重程度）
2. 已修改文件
3. 尚未修改但建议尽快处理的问题
4. 剩余风险
5. 编译结果
6. 建议优先复核的位置
```
