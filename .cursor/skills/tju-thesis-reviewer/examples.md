# Examples

## 示例 1：格式初审

### 用户输入

请按天津大学本科毕业论文规范审查这份论文，重点看前置页、页码、页眉和标题格式。

### 预期动作

1. 检查前置页顺序与完整性。
2. 检查罗马页码和正文页码切换。
3. 检查页眉文本、标题层级和段落版式。
4. 输出高优先级问题并直接修改能落地的问题。

## 示例 2：图表专项 review

### 用户输入

请重点检查图表，看看题注位置、三线表、图中文字和图注语气是否符合学校要求。

### 预期动作

1. 检查图下表上的题注位置是否统一。
2. 检查表格是否为三线表，必要时重排宽表。
3. 检查图中文字大小、留白和可读性。
4. 把图注和表注改成“口径 + 来源”的送审语气。

## 示例 3：送审前总检

### 用户输入

请做送审前最后一轮总检，给我一个可直接汇报给导师的问题清单。

### 预期输出结构

1. 问题清单（按严重程度）
2. 已修改文件
3. 尚未修改但建议尽快处理的问题
4. 剩余风险
5. 编译结果
6. 建议优先复核的页面或章节
